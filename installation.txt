이 스크립트를 실행하기 위한 설치 및 설정 안내서입니다.

## 1. 사전 요구사항

- **Python 3**: 시스템에 Python 3가 설치되어 있어야 합니다.
- **Google Chrome**: Google Chrome 브라우저가 설치되어 있어야 합니다. `webdriver-manager`가 자동으로 버전에 맞는 드라이버를 설치하지만, 브라우저 자체는 필요합니다.

## 2. 필요한 Python 라이브러리 설치

아래 명령어를 터미널(명령 프롬프트, PowerShell 등)에서 실행하여 필요한 라이브러리를 설치합니다.
`requirements.txt` 파일이 같은 폴더에 있는 경우, 아래 명령어로 한 번에 설치할 수 있습니다.

```
pip install -r requirements.txt
```

만약 `requirements.txt` 파일을 사용하지 않는다면, 아래 명령어로 개별 라이브러리를 설치해주세요.

```
pip install selenium webdriver-manager
```

## 3. 환경 변수 설정

스크립트는 보안을 위해 네이버 아이디와 앱 비밀번호를 환경 변수에서 읽어옵니다.

**중요**: 네이버 로그인의 경우, 2단계 인증을 사용한다면 일반 비밀번호가 아닌 **앱 비밀번호**를 발급받아 사용해야 합니다.

### Windows (명령 프롬프트)
```
set NAVER_ID="YOUR_NAVER_ID"
set NAVER_APP_PASSWORD="YOUR_NAVER_APP_PASSWORD"
```

### Windows (PowerShell)
```
$env:NAVER_ID="YOUR_NAVER_ID"
$env:NAVER_APP_PASSWORD="YOUR_NAVER_APP_PASSWORD"
```

**참고**: 위 명령어는 현재 터미널 창에서만 유효합니다. 컴퓨터를 재부팅해도 유지되게 하려면 '시스템 환경 변수'에 직접 추가해야 합니다.

## 4. 스크립트 실행

모든 설정이 완료되면 아래 명령어로 스크립트를 실행합니다.

```
python mail_copy.py
```
